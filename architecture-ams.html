Le défi : scaler la sécurité réseau

Concevoir un réseau pour 2100 équipements, ce n'est pas juste "brancher des câbles". Chaque décision d'architecture a des implications de sécurité, de performance et de maintenabilité. Un réseau plat (flat network) à cette échelle est une catastrophe : une seule machine compromise peut latéralement atteindre tout le réseau.

La solution : une architecture en couches (three-tier) avec segmentation par VLAN et filtrage ACL strict entre les segments.

Plan de segmentation VLAN

Chaque département dispose de son propre VLAN isolé. La communication inter-VLAN est contrôlée par des ACL strictes sur les switches de distribution.

Configuration Cisco IOS

! Configuration VLAN et routage inter-VLAN
! Switch Distribution — Bâtiment A

vlan 10
 name MANAGEMENT
vlan 30
 name UTILISATEURS

! Interface SVI pour routage inter-VLAN
interface Vlan10
 ip address 10.10.10.1 255.255.255.0
 ip access-group ACL_MGMT in
 no shutdown

interface Vlan30
 ip address 10.10.30.1 255.255.254.0
 no shutdown

! Configuration du trunk vers le core
interface GigabitEthernet0/1
 switchport mode trunk
 switchport trunk allowed vlan 10,20,30,40,50
 switchport trunk native vlan 99
! ACL — Isolation du VLAN MANAGEMENT
! Seuls les admins (10.10.10.0/24) accèdent aux équipements

ip access-list extended ACL_MGMT
 ! Autoriser SSH depuis le réseau admin
 permit tcp 10.10.10.0 0.0.0.255 any eq 22
 ! Autoriser SNMP pour monitoring
 permit udp 10.10.10.0 0.0.0.255 any eq 161
 ! Bloquer tout le reste vers le MGMT
 deny   ip any 10.10.10.0 0.0.0.255 log
 permit ip any any

! ACL — Isolation VLAN INVITÉS (Internet only)
ip access-list extended ACL_GUEST
 deny   ip 192.168.100.0 0.0.0.255 10.10.0.0 0.0.255.255
 permit ip 192.168.100.0 0.0.0.255 any

Sécurité au niveau des ports d'accès

Sur chaque switch d'accès, la fonctionnalité Port Security limite le nombre d'adresses MAC par port à 1, avec une action de violation en mode shutdown. Cela empêche la connexion d'équipements non autorisés.

! Configuration Port Security sur les ports utilisateurs
interface range FastEthernet0/1 - 24
 switchport mode access
 switchport access vlan 30
 ! Activer Port Security
 switchport port-security
 switchport port-security maximum 1
 switchport port-security violation shutdown
 switchport port-security mac-address sticky
 ! Protection contre les attaques STP (BPDU Guard)
 spanning-tree portfast
 spanning-tree bpduguard enable

Architecture validée avec un temps de convergence OSPF < 2 secondes et zéro communication non autorisée entre VLANs lors des tests de pénétration réseau.
