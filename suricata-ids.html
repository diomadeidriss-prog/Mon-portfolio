Contexte du projet

Un réseau non surveillé est une porte ouverte. L'objectif de ce projet était de déployer une sonde réseau complète capable de surveiller le trafic en temps réel, d'identifier les comportements anormaux et de bloquer automatiquement les attaques avérées.

Suricata est un moteur IDS/IPS open-source haute performance, utilisé en production par des entreprises du monde entier. Sa capacité à analyser le trafic à la vitesse du réseau (multi-threading) en fait un outil de référence.

Architecture déployée

La sonde est positionnée en coupure sur l'interface réseau principale de la machine, en mode inline IPS via NFQueue — ce qui permet non seulement de détecter, mais aussi de bloquer les paquets malveillants en temps réel.

Configuration & règles personnalisées

Au-delà des rulesets communautaires (Emerging Threats, Snort Community), j'ai rédigé mes propres règles pour détecter des comportements spécifiques à notre environnement de test.

# Détection de scan de ports SYN (Nmap -sS)
alert tcp any any -> $HOME_NET any (
    msg:"SCAN SYN Port Scan détecté";
    flags:S;
    threshold: type threshold, track by_src,
               count 20, seconds 2;
    classtype:network-scan;
    sid:1000001; rev:1;
)

# Détection bruteforce SSH
alert tcp any any -> $HOME_NET 22 (
    msg:"BRUTEFORCE SSH tentative multiple";
    flow:to_server,established;
    threshold: type both, track by_src,
               count 5, seconds 60;
    classtype:attempted-admin;
    sid:1000002; rev:1;
)

# Détection de C2 — beacon régulier
alert http $HOME_NET any -> $EXTERNAL_NET any (
    msg:"MALWARE Possible C2 beacon";
    flow:established,to_server;
    content:"User-Agent|3a 20|Go-http-client";
    sid:1000003; rev:1;
)

Analyse des logs — EVE JSON

Suricata génère ses alertes au format EVE JSON, un format structuré qui facilite l'intégration dans des SIEM. J'ai écrit un script Python pour parser, filtrer et visualiser ces logs, et générer des rapports quotidiens.

import json
from collections import Counter

def parse_eve_log(filepath):
    alerts = []
    with open(filepath) as f:
        for line in f:
            event = json.loads(line)
            if event.get("event_type") == "alert":
                alerts.append({
                    "timestamp": event["timestamp"],
                    "src_ip": event["src_ip"],
                    "signature": event["alert"]["signature"],
                    "severity": event["alert"]["severity"]
                })
    # Top 10 IPs sources les plus actives
    top_ips = Counter(a["src_ip"] for a in alerts).most_common(10)
    return alerts, top_ips

Tests & validation

Pour valider les règles, j'ai simulé des attaques dans un environnement isolé :

  • Scan de ports avec Nmap (SYN scan, version detection, OS fingerprinting)
  • Bruteforce SSH avec Hydra
  • Exploitation de failles web via SQLmap
  • Simulation de trafic malveillant avec Metasploit
